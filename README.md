# E-Commerce Web Application (Backend)

## Overview
This project is a backend system for an e-commerce application developed using Flask. It handles product management, user sessions, and cart functionality with database integration.

## Key Features
- Product listing and management
- User session handling
- Cart operations including add, remove, and quantity updates
- Role-based access for admin and users
- Image upload and retrieval support

## Tech Stack
- Python
- Flask
- MySQL
- REST APIs

## Responsibilities
- Developed Flask APIs for product and cart operations
- Designed relational database tables for users, products, and carts
- Implemented access control for admin and standard users
- Integrated image handling using Flask static files

## How to Run
1. Clone the repository
2. Install dependencies using `pip install -r requirements.txt`
3. Configure MySQL database
4. Run the Flask application
